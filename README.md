# 🚀 Blinkit Sales Analysis (SQL + Power BI Project)

## 📌 Project Overview

This project focuses on analyzing **Blinkit (online grocery delivery platform)** sales data to extract meaningful business insights. The analysis is performed using **SQL for data querying** and **Power BI for interactive visualization**.

The project helps in understanding customer behavior, sales trends, and key revenue drivers for better decision-making.

---

## 🎯 Business Objectives

* Analyze overall sales performance 📊
* Identify top-performing product categories 🛒
* Understand customer purchasing behavior 👥
* Track revenue and profit trends 📈
* Generate actionable business insights

---

## 🧰 Tools & Technologies

* **SQL** (Data Extraction & Analysis)
* **Power BI** (Dashboard & Visualization)
* **Excel / CSV Dataset**

## 🔄 Data Analysis Workflow

1. **Data Collection**

   * Imported raw dataset (CSV/Excel)

2. **Data Analysis using SQL**

   * Data filtering and aggregation
   * Grouping and summarization
   * KPI calculations

3. **Data Visualization (Power BI)**

   * Built interactive dashboard
   * Created measures using DAX

4. **Insights & Business Recommendations**

---

## 📊 Key Insights

* 📌 A small number of categories generate the majority of sales
* 📌 Customer purchasing patterns vary across segments
* 📌 Certain regions contribute more to overall revenue
* 📌 Sales trends show variation over time
* 📌 Popular products drive repeat purchases
